# Higgsfield-Style Storyboard / References / AI Video Workflow Research Package

This zip contains a PRD/spec research handoff for adapting Higgsfield-style AI video production patterns into Open-Generative-AI.

## Files

- `research_report.md` — full 12-section research report.
- `evidence_table.csv` — source URL, claim, confidence, relevance, notes.
- `higgsfield_ux_map.md` — confirmed/inferred UX map and OGA translation.
- `prior_art_scan.csv` — repositories/tools, license status, maturity, reuse recommendation.
- `recommended_product_model.md` — product object model and behavior principles.
- `recommended_backend_data_model.md` — storage, entity, migration, rollback guidance.
- `recommended_frontend_ux.md` — sidebar, shot board, modals, references, variants, failure states.
- `two_phase_prd.md` — two-phase PRD outline.
- `acceptance_criteria.md` — spec acceptance criteria.
- `test_verification_checklist.md` — regression and feature verification checklist.
- `login_followup_plan.md` — legal browser/Firecrawl/account validation plan.
- `implementation_handoff_notes.md` — planner notes for likely repo surfaces.
- `sample_project_schema.json` — example metadata-only project sidecar.
- `source_urls.txt` — source URL list.

## Main recommendation

Start with a metadata-only Project → Scene → Shot board and References library linked to existing `.native-media` jobs/assets/uploads. Skip identity training, realtime collab, community, MCP, and full timeline editing until the core storyboard/reference workflow is stable.
